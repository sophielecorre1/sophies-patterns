Intent

This pattern allows an application to dynamically select a behavior based on two runtime types and a contextual behavior, replacing the Visitor pattern or nested conditionals.

Structure

A matrix maps a triplet (Type A, Type B, Behavior) to an associated function to execute.

Advantages

  • Readable, testable, and declarative
  • Respects the Open/Closed Principle
  • Avoids Visitor complexity, switch, and instanceof

Java Example

matrix.register(Cat.class, Fish.class, BehaviorType.EAT,
    (cat, fish) -> "The cat eats the fish.");

matrix.apply(new Cat(), new Fish(), BehaviorType.EAT);
